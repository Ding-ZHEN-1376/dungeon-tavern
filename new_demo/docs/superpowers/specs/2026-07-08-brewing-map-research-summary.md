# 酿酒系统当前设计总结

本文总结当前关于酿酒研发系统的设计思路。系统目标是把酿酒从简单的材料列表选择，改造成一个以“酒谱地图、材料路线、处理方法、迷雾探索”为核心的小型解谜玩法。

## 1. 核心定位

酿酒系统分为两层：

- 已完全解锁的配方：玩家可以直接酿造，不需要重复游玩研发小游戏。
- 尚未完全解锁的大配方：玩家需要进入酒谱地图，通过投放材料和处理材料来探索并解锁具体酒水。

大配方不是单一酒，而是一个系列酒谱。一个大配方中可以隐藏多种具体酒。玩家通过多次研发逐步发现这些酒，全部发现后，该大配方进入可直接酿造状态。

## 2. 大配方与酒谱地图

每个大配方对应一张独立地图。地图大小由该配方决定，但后续设计统一使用奇数尺寸，方便设置中心起点。

首版地图设计：

- 大配方名称：地牢蘑菇酒谱
- 地图大小：11x11
- 坐标范围：`(0,0)` 到 `(10,10)`
- 起点：地图中心 `(5,5)`
- 隐藏酒数量：2 种
- 每种酒的区域大小：2x2
- 酒 A 左上角：`(1,1)`
- 酒 B 左上角：`(8,8)`

酒的区域不是一个点，而是 2x2 区块。玩家的指示物只要进入某个酒区块，就算研发成功并发现该酒。

## 3. 材料限制

每个大配方会限制本次研发可使用的材料。这样可以保证每个酒谱有明确主题，也避免玩家被大量材料选项淹没。

以“地牢蘑菇酒谱”为例，可用材料先限制为：

- 水
- 黑麦
- 洞穴红蘑菇
- 洞穴蓝蘑菇

后续可以为其他大配方设计不同材料组。例如矿石酒谱可以限制为泉水、苔藓、矿石类材料；草药酒谱可以限制为清水、小麦、草药类材料。

## 4. 投料次数与路线规则

每次研发有投料次数上限。首版建议：

- 每次研发最多投入 5 次材料。
- 一种材料可以重复投入。
- 每次投入都会让地图上的指示物从当前位置沿材料路线移动。
- 路线确认后不可撤回。
- 如果路线会走出地图边界，本次投入不能确认。

这里的“5 次”指的是最多 5 次投料动作，不是最多移动 5 个格子。因为从中心 `(5,5)` 到两个 2x2 酒区块的最近距离都超过 5 格，需要通过多次材料路线组合到达。

## 5. 材料路线

材料代表基础移动路线。玩家选择材料后，系统会从当前指示物位置预览该材料的路线。

首版基础路线：

| 材料 | 基础路线 |
| --- | --- |
| 洞穴红蘑菇 | 上 |
| 洞穴蓝蘑菇 | 左 |
| 普通水 | 无移动，但作为初始材料，必须放入 |
| 普通黑麦 | 无移动，但作为初始材料，必须放入 |

从中心 `(5,5)` 出发，红蘑菇主要帮助玩家向上移动，蓝蘑菇主要帮助玩家向左移动。通过处理方法，材料路线可以被延伸、旋转或反转，从而到达不同方向的酒区块。

## 6. 处理方法

处理方法不应写死为每个材料的独立路线，而应作为对材料基础路线的规则变换。这样以后增加新材料时，只需要定义基础路线，处理方法就能复用。

当前设计保留一类处理方法：

| 处理类型 | 暂定名称 | 效果 |
| --- | --- | --- |
| A | 研磨 / 捣碎 | 延伸当前材料路线，方向不变，距离增加 |
| B | 烘烤 / 调向 | 以 5x5 中心为基准，将路线顺时针旋转 90 度 |


## 7. 处理道具

处理方法需要对应道具。玩家拥有道具后，才可以使用对应处理方法。

首版道具设定：

- 研磨器：解锁研磨 / 捣碎。
- 烘烤台：解锁烘烤 / 调向。

道具可以从商店购买。这样商店和经营成长会直接影响酿酒研发能力：玩家买到新工具后，同一批材料会出现新的路线解法。

UI 中应该显示材料支持的处理方法，但玩家没有对应道具时，该处理选项显示为锁定或灰色，不可选择。

## 8. 首版路线表

以下是“地牢蘑菇酒谱”的首版路线示例：

| 材料与处理 | 路线 |
| --- | --- |
| 洞穴红蘑菇 | 上，上 |
| 洞穴蓝蘑菇 | 左，左 |
| 洞穴红蘑菇 + 研磨 | 上，上，上 |
| 洞穴蓝蘑菇 + 研磨 | 左，左，左 |
| 洞穴红蘑菇 + 烘烤 | 右，右 |
| 洞穴蓝蘑菇 + 烘烤 | 下，下 |
| 洞穴红蘑菇 + 烘烤 + 研磨 | 右，右，右 |
| 洞穴蓝蘑菇 + 烘烤 + 研磨 | 下，下，下 |


## 9. 单次路线范围限制

每个材料处理后的路线都应限制在以当前点为中心的 5x5 范围内。

含义：

- 单次投料的横向偏移不能超过 2 格。
- 单次投料的纵向偏移不能超过 2 格。
- 单次投料不会让玩家跨越过大距离。
- 玩家需要通过多次材料组合来接近目标酒区块。

这个限制能避免某个材料或处理方法过强，也能让“最多 5 次投料”的限制更有意义。

## 10. 投入交互流程

投入流程不是“先选材料，再弹出处理方法”。材料和处理方法应该在同一个界面中同时选择。

推荐流程：

1. 玩家打开一个尚未完全解锁的大配方。
2. 系统显示该配方的酒谱地图。
3. 指示物出现在地图中心 `(5,5)`。
4. 右侧显示当前酒谱允许使用的材料。
5. 玩家选中某个材料。
6. 同一界面显示该材料可用的处理方法（可以两个都选）。
7. 玩家选择处理方法。
8. 地图从当前位置显示“材料 + 处理方法”对应的路线预览。
9. 如果玩家缺少处理道具，处理方法不可选。
10. 如果预览路线越界，确认按钮不可用。
11. 玩家点击“确认投入”。
12. 材料被消耗，路线被锁定。
13. 指示物沿路线移动到末端。
14. 系统揭示路径经过位置及终点附近的迷雾。
15. 玩家继续下一次投料，直到玩家走到酒的区域里。
16. 玩家解锁当前酒的配方，配方依据本次实验的过程。

确认投入前，玩家可以自由切换材料和处理方法来查看不同路线。确认后，本次路线不能再修改。

## 11. 迷雾与发现

酒谱地图带有迷雾。迷雾的作用是让研发有探索感，但不能让玩家完全盲猜。

当前思路：

- 玩家选择大配方后，可以揭示起点附近区域。
- 玩家确认投入后，路径经过的格子会被揭示。
- 研发结束后，可以揭示终点附近区域。
- 路线预览只显示将要走的路径，不提前揭示迷雾中的内容。

这样玩家能预判路线，但仍然需要通过实际投入来探索地图。

## 12. 研发成功与失败

成功条件：

- 指示物进入某个隐藏酒的 2x2 区块。
- 该酒被记录到大配方中。
- 后续可以使用记录下来的路线作为配方线索。

失败条件：

- 投料次数耗尽仍未进入任何酒区块。
- 玩家主动结束研发但没有进入酒区块。

失败结果：

- 产出一份失败酒。
- 失败酒带有负面标签。
- 研发过程仍然保留地图揭示和路线记录。

失败酒不是纯惩罚。它可以低价出售，也可以用于后续任务或特殊顾客需求。这样失败仍然有资源和信息价值。

## 13. 失败酒与负面标签

失败酒用于承接研发失败。

示例：

```text
浑浊蘑菇酒
标签：蘑菇、浑浊、苦涩、发酵不足
```

负面标签可以来自：

- 路线没有进入酒区块。
- 路线经过危险区域。
- 使用了不适合当前酒谱目标的处理方法（需要限制烘烤和研磨的次数？）。
- 投料次数过多或路线过长。

首版可以只做一种通用失败酒，后续再根据失败位置、材料组合和危险区域生成不同失败酒。

## 14. 危险区域与特殊区域

危险区域和特殊区域是后续可以加入的地图机制。

首版建议只保留非常少的特殊格，避免玩家过早理解过多规则。

可选设计：

- 危险区：路径经过后，成品增加负面标签，例如浑浊、焦糊、腐败。
- 流向区：路径经过后，指示物额外偏移 1 格。
- 阻挡区：路线不能穿过，必须绕行。

第一版最多只加入一种危险区。等基础路线玩法成立后，再逐步增加地图地形。

## 15. 最优解

研发成功不代表玩家已经找到最优解。

首版目标：

- 第一次：能走进酒区块，发现酒。
- 后续：用更低成本、更少投料、更少负面影响的路线复现该酒。

最优解不应只看材料数量，而应综合成本。

推荐计算方向：

```text
配方成本 = 材料数量 + 材料稀有度成本 + 处理成本 + 危险区惩罚
```

这样可以避免“一个稀有材料走很远”被系统误判为最优，也能保留经营上的资源取舍。

## 16. 与拼图想法的关系

早期曾考虑让材料对应拼图形状，玩家用材料拼配方壳子。当前设计中，拼图系统不再作为主玩法。

保留的部分是：

- 材料处理会改变材料的路线形态。
- 不同处理方式使同一种材料产生不同路线。

也就是说，拼图思路被转化为“路线变化”思路。主系统保持统一：玩家始终是在酒谱地图上选择材料、选择处理、预览路线、确认移动。

## 17. 首版功能限制

首版应严格限制范围：

- 只做 1 张大配方：地牢蘑菇酒谱。
- 地图固定为 11x11。
- 起点固定为 `(5,5)`。
- 只隐藏 2 种酒。
- 每种酒为 2x2 区块。
- 只做洞穴红蘑菇和洞穴蓝蘑菇。
- 只做水和黑麦作为基础材料。
- 处理方法先做不处理、研磨、烘烤。
- 处理道具先做研磨器和烘干器 / 烤炉。
- 投料上限为 5 次。
- 失败酒先做一种通用版本。
- 危险区域可以暂不做，或只做一种。

这个范围足够验证核心玩法，不会过早进入大系统膨胀。

## 18. 简要文字版游玩流程

玩家在酒馆中获得一张“地牢蘑菇酒谱”。这张酒谱还没有完全解锁，里面隐藏着两种蘑菇酒。

玩家来到酿酒台，选择研发“地牢蘑菇酒谱”。界面打开后，出现一张 11x11 的酒谱地图，地图中央 `(5,5)` 有一个指示物。地图大部分区域被迷雾遮住，只能看到起点附近。

右侧材料栏显示本酒谱允许使用的材料：水、黑麦、洞穴红蘑菇、洞穴蓝蘑菇。玩家点击洞穴红蘑菇，并在同一界面选择“不处理”。地图上出现一条从当前位置向上的路线预览。玩家觉得路线合适，点击“确认投入”。红蘑菇被消耗，指示物沿路线向上移动一格，路径经过的迷雾被揭开。

接着，玩家选择洞穴蓝蘑菇，并选择“研磨”。因为玩家已经拥有研磨器，所以研磨选项可用。地图预览显示路线会向左移动两格。玩家确认投入，指示物向左移动两格。

玩家不断根据当前位置、可用材料和路线预览选择下一次投入。每次投入前都可以查看不同材料和处理方法产生的路线，但一旦确认投入，本次路线就锁定，不能撤回。

如果玩家在最多 5 次投料内让指示物进入左上角或右下角的 2x2 酒区块，就成功发现一种新酒。该酒会记录到“地牢蘑菇酒谱”中。等酒谱里的所有酒都被发现后，玩家以后就可以直接酿造这些酒，不必每次重新走地图。

如果玩家 5 次投料后仍然没有进入任何酒区块，本次研发失败，但不会完全归零。系统会产出一份带负面标签的失败酒，例如“浑浊蘑菇酒”。同时，本次走过的路线和揭开的地图会保留下来，帮助玩家下一次研发时判断该往哪里走。
